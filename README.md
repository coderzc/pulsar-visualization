# Pulsar 消息可视化工具

## 功能

- [x] 模拟消息的生产和消费
- [ ] 模拟消息的延迟
- [ ] 模拟消息的重复
- [ ] 模拟消息的顺序
- [ ] 模拟消息的压缩


## 使用

1. 下载代码
2. 运行 `npm install`
3. 运行 `npm run dev`
4. 打开 `http://localhost:3000` 查看效果
